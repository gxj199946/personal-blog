hugo博客添加字数统计和阅读需要xx分钟的功能

Posted by Puppet on Wed, Nov 30, 2022

给hugo博客添加字数统计和阅读需要xx分钟的功能

修改于2021-02-16 20:01:56阅读 5460

缘由

在看别人博客时经常在文章标题下面见到有共xx字,阅读约xx分钟的字眼，这个功能在hugo中本身就已经自带了，开启这个功能即可。

修改博客页面

页面位于对应主题目录下的：layouts/_default/single.html,搜索subtitle可以找到以下类似的代码块:

1<h2 class="subheading">{{ .Params.subtitle }}</h2>
2                    <span  class="meta">Posted by {{ if .Params.author }}{{ .Params.author }}{{ else }}{{ .Site.Title }}{{ end }} on {{ .Date.Format "Monday, January 2, 2006" }}
3                        {{ if .Site.Params.page_view_conter }}
4                        {{ partial "page_view_counter.html" . }}
5                        {{ end }}
6                    </span>

复制

修改后是一下的样子，其实就是引入两个变量:

  1. WordCount 文章字数
  2. ReadingTime 阅读时间
1                    <h2 class="subheading">{{ .Params.subtitle }}</h2>
2                    <span  class="meta">Posted by {{ if .Params.author }}{{ .Params.author }}{{ else }}{{ .Site.Title }}{{ end }} on {{ .Date.Format "Monday, January 2, 2006" }}
3                        {{ if .Site.Params.page_view_conter }}
4                        <span id="busuanzi_container_page_pv">|<span id="busuanzi_value_page_pv"></span><span>
5                            {{ partial "page_view_counter.html" .  }}
6                            阅读 </span></span>|<span class="post-date">共{{ .WordCount  }}字</span>，阅读约<span class="more-meta"> {{ .ReadingTime  }} 分钟</span>
7                        {{ end }}
8                    </span>

复制

修改配置文件

在修改了上述内容后主题已经支持字数统计和阅读时间的功能，下面需要在博客的配置文件当中启动这个功能。

找到config.toml文件中的page_view_conter，配置为true即可。

1page_view_conter=true

在重新打开任意一篇Blog后发现已经可以看到这个功能的展示了。
